Latest Release: Monkey v1.6.0 August 10th, 2015

Monkey Project development team is glad to announce the availability of Monkey v1.6.0 release, for more information about this new version we invite you to read the official Release Notes.

Start with Monkey in less than 30 seconds

{% highlight bash %} $ wget http://monkey-project.com/releases/1.6/monkey-1.6.0.tar.gz $ tar zxfv monkey-1.6.0.tar.gz $ cd monkey-1.6.0 $ ./configure --malloc-libc $ make $ build/monkey {% endhighlight %}
